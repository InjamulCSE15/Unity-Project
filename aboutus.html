Services

  • This project provides a way to allocate exam hall for every students without any clash.
  • The system stores the list of class and its capacity where exam can be conducted and list of students appearing the exam on each day .
  • The system also provides a facility to take printout of students in each center.
  • It also generates report of students list for each room and master report. It also generates invigilator shift-wise duty chart report.
  • The main aim of this project is to make an android app to know the seating of the exam hall for the student.
  • And also admin can easily allocate hall as per the seating easily.
  • Paper work has been reduced for allocating the seat in exam halls.
  • Admin adds all the students’ details and by using these details halls is allocated for the exam.
  • Based on the capacity of the hall seating will be allocated.
  • Students can easily see their hall for the exam easily.
  • Updating of information becomes so easier.
  • System security ,Data Security and reliability are the striking features.
  • It provider a friendly graphical user interface which proves to be best.
  • The system has adequate scope for modification in future if it is necessary.
